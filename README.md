## Overview

**SEONG CHIHYUN Portfolio** is a personal web portfolio designed to introduce who I am, what I build, and how I approach software development.

This website is intended to work as a clean and simple personal homepage where visitors can quickly understand my projects, skills, interests, and contact information.

> Goal: create a portfolio that feels simple, modern, readable, and developer-focused.

## Design Direction

This portfolio follows a minimal and modern design style.

### Design Keywords

- Minimal
- Clean
- Developer-focused
- Responsive
- Readable
- Project-centered
- Simple but polished

### Visual Style

The design should feel like a calm personal portfolio rather than a complicated landing page.

Recommended direction:

- Light background with strong contrast text
- Simple hero section
- Clear project cards
- Smooth spacing between sections
- Consistent typography
- Subtle hover animations
- Mobile-first responsive layout
